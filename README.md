# ZeroFux

> A stateless unidirectional data flow implemented with [Custom Events](https://developer.mozilla.org/de/docs/Web/API/CustomEvent).

No Flux plus no Redux equals ZeroFux. 

## Install

```sh
npm i zero-fux
```
